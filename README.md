# Quantum-Safe Zero Trust AWS Framework

This project builds a realistic enterprise-grade cloud security environment featuring:

- Zero Trust AWS VPC (private-only)
- IAM Zero Trust identity model
- AWS ACM Private Certificate Authority (PKI)
- Automated TLS certificate rotation
- Post-Quantum Cryptography hybrid key integration (Kyber + RSA)
- Security drift scanner (IAM, SGs, encryption, TLS)
- Auto-remediation Lambda functions
- Terraform Infrastructure-as-Code
- Full compliance reporting pipeline

This repository follows enterprise IaC structures with modules for VPC, IAM, PKI, Lambda automation, PQC, and Zero Trust security baselines.
